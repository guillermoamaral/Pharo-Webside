# Pharo-Webside
Webside API implementation in Pharo

## Basic Installation

You can load **Teapot** evaluating:
```smalltalk
Metacello new
	baseline: 'Webside';
	repository: 'github://guillermoamaral/Pharo-Webside:main';
	load
```
>  Change `main` to some released version if you want a pinned version.
